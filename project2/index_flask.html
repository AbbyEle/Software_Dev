Project 2: Abbie's WigStore

This is a Flask-based web application that runs on a local Python server.

GitHub Pages only supports static websites, so this project can't run directly here.

To Run Locally:

git clone https://github.com/AbbyEle/Software_Dev.git
cd Software_Dev/project2
pip install -r requirements.txt
flask run

Check out the project repository for more details.
